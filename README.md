# OptimizationMethods
1 курс (Магистратура), Методы оптимизации:
- метод дихотомии;
- метод золотого сечения;
- метод Фибоначчи;
- метод прямого покоординатного поиска;
- метод Хука-Дживса;
- метод Нелдера-Мида.  
1 и 2 практические работы (Задание 1.11, 1.12. Задание 2.11) (Язык - С#, интерфейс - Windows Forms):
